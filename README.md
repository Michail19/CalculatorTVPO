# Назначение программы

Программа «Финансовый Калькулятор» представляет собой финансовый калькулятор с функцией конвертации валют. Он позволяет выполнять базовые арифметические операции (сложение, вычитание, умножение, деление, вычисление процента) и мгновенно конвертировать суммы между различными валютами по актуальному курсу.

---

# Системные требования

* Операционная система: Windows 10 или новее
* Память (ОЗУ): 1 ГБ или более
* Свободное место на диске: 512 МБ
* Интерпретатор Python: версия 3.9 или выше
* Библиотеки: PyQt5, requests
* Интернет-соединение: Требуется для первоначальной загрузки и обновления курсов валют.

---

# Установка и запуск

Убедитесь, что на вашем компьютере установлен Python версии 3.9 или выше.

Установите необходимые библиотеки, выполнив в командной строке (CMD или PowerShell):

```bash
pip install PyQt5 requests
```

Сохраните файл с кодом программы (например, main.py) в удобную для вас папку.

Запустите программу, выполнив в командной строке команду:

```bash
python путь_к_файлу/main.py
```

Или двойным кликом по файлу, если ассоциации файлов в системе настроены правильно.

# Описание интерфейса

Главное окно программы состоит из двух основных частей: панели конвертера валют и панели калькулятора.

1. Панель конвертера валют:

* Три строки ввода/вывода: Каждая строка состоит из выпадающего списка для выбора валюты и поля для ввода/вывода числового значения.
* Выпадающий список: Позволяет выбрать валюту из списка (например, RUB, USD, EUR). Одну и ту же валюту нельзя выбрать в двух разных строках одновременно.
* Поле ввода: Предназначено для ввода числовых значений. Активное поле выделено курсором. Изменение значения в любом поле автоматически пересчитывает и отображает эквивалент в других валютах.

2. Панель калькулятора:

* Цифровые кнопки (0-9, 00): Для ввода чисел.
* Кнопки операций (+, -, , /, %): Для выбора арифметической операции.
* Кнопка десятичного разделителя (.)
* Кнопка очистки (C): Сбрасывает текущее значение в активном поле ввода до «0».
* Кнопка удаления (⌫): Удаляет последний введенный символ в активном поле ввода.
* Кнопка равенства (=): Выполняет вычисление введенного арифметического выражения в активном поле ввода. Также вычисление можно запустить клавишей Enter.

# Порядок работы

* Конвертация валют:

Убедитесь, что программа загрузила актуальные курсы валют (проверяется по наличию в списках валют, отличных от **USD**).

Выберите с помощью выпадающих списков валюту для каждой строки.

Введите сумму в любое поле ввода. Суммы в других полях автоматически пересчитаются.

Для расчета от другой суммы просто введите новое значение в нужном поле.

* Арифметические вычисления:

Щелкните мышью по любому из трех полей ввода, чтобы сделать его активным. Все вводимые с клавиатуры или нажимаемые на экране цифры и операторы будут поступать в это поле.

Введите математическое выражение с помощью кнопок на экране или клавиатуры (поддерживаются цифры, основные операторы **+**, **-**, *, **/**, **%**, точка **.**, клавиша **Backspace** и **Enter** для вычисления).

Нажмите кнопку **=** или клавишу **Enter** на клавиатуре, чтобы вычислить результат выражения. Результат отобразится в активном поле.

После вычисления результат автоматически будет конвертирован в другие валюты.

* Копирование результата:

Чтобы скопировать результат вычислений, выделите текст в поле ввода с помощью мыши и нажмите Ctrl+C или выберите в контекстном меню пункт «Копировать».

# Возможные проблемы и решения

«Ошибка загрузки курсов» в консоли: Программа не смогла подключиться к API для получения курсов. Проверьте подключение к интернету. Программа использует заранее заданные базовые курсы (USD, RUB, KZT, EUR).

Курсы валют не обновляются: Программа загружает курсы только один раз при запуске. Для обновления курсов перезапустите программу.

Нельзя выбрать одну и ту же валюту дважды: Это ограничение программы для исключения бессмысленных конвертаций. Выберите разные валюты для каждого поля.

Поле ввода показывает «Ошибка»: Введенное математическое выражение содержит синтаксическую ошибку (например, два оператора подряд 10++5). Нажмите C для очистки поля и введите выражение заново.
